# Java demos

This is Java project using Maven as build tool.

The goal is to provide simple explanation and examples for all Java JEPs to teach new developers.

Tooling is described in `README.md` in `Prerequisites` section.

Usage is described in `README.md` in `Usage` section.

Project structure and typical workflow for adding new demos is described in `CONTRIBUTING.md`. When prompted to create new demo, read `CONTRIBUTING.md` for instructions. Use `src/main/java/org/javademos/java26/jep526/LazyConstantsDemo.java` as baseline implementation. Make sure the project compiles and runs without errors.
